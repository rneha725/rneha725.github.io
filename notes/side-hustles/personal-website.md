# Personal Website

The goal: a place on the internet that is actually mine. Not a portfolio template. Not a LinkedIn clone. Something that reflects how I actually think.

## Constraints (the good kind)

- Everything written in Markdown — if it can't be a plain text file, it doesn't belong here
- No framework, no build step — just a GitHub repo and a browser
- Publishing feels like journaling, not deploying

## What it should do

- Showcase writing, notes, and occasional projects
- Organize content the way I think — by what it *is*, not by date
- Load fast. Look clean. Feel like me.

## Current status

The design is working. Setting up the folder structure and file system now.
