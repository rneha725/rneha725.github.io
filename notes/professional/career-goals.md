# Career Goals 2026

Not a rigid plan. A direction. Reviewed quarterly, adjusted honestly.

## Technical

- Go deeper on systems design — not just knowing the patterns, but understanding the trade-offs
- Write code that other people can read without explanations
- Get comfortable with data pipelines and working at scale

## Communication

- Learn to explain complex things simply (this is a skill, not a personality trait)
- Write more in public — notes like these count
- Say no to things that don't move the needle

## Theme for this year

**Build things that last.**

Not the flashiest thing, not the fastest thing — the thing that's still working and still useful a year from now.
